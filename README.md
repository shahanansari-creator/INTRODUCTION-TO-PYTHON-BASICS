## 🧠 OOP at a Glance

The OOP section is one of the most detailed parts of these notes:

```python
# Inheritance + Polymorphism Example
class Animal:
    def speak(self): return "Some sound"

class Dog(Animal):
    def speak(self): return "Woof! 🐶"

class Cat(Animal):
    def speak(self): return "Meow! 🐱"

for animal in [Dog(), Cat()]:
    print(animal.speak())
```

**Four pillars covered in depth:**

```
┌─────────────────────────────────────────────────────┐
│  🔒 Encapsulation  →  Hide data using __private      │
│  🧬 Inheritance    →  Child inherits from Parent     │
│  🎭 Polymorphism   →  Same method, different output  │
│  🎯 Abstraction    →  Hide complexity via ABC module  │
└─────────────────────────────────────────────────────┘
```
